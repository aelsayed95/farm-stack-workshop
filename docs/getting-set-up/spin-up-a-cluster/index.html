Spin Up a Cluster

Once you have a MongoDB Atlas account, and you're logged in, click on the "Create" button. It'll take you to a page to configure your new MongoDB cluster.

If you haven't created a cluster before, the Create button, or something like it, should be in the middle of the screen after you log in.

If you have already created a cluster (go you!) then it'll be on the right-hand side, above any clusters that you've already spun up. In this case, you could use the existing cluster. Just make sure that the database name we configure doesn't conflict with any databases you've already created.

Configure your cluster

Select the "Shared" tab at the top of the form, and then select a region that's near you. I chose "Ireland" because that was my nearest location when I wrote this.

Make sure your cluster tier is M0 Sandbox - that's the free tier. At this point you can name your cluster. You won't get another chance! The name can just be something useful for you. It doesn't need to be unique.

Now click the big green "Create Cluster" button, and jump to the next page while you wait for your cluster to be configured for you. It usually takes 3-7 minutes, but you won't need it for a little while.
